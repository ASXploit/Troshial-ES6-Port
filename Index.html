Init Library

let game = new Troshial();

Creating the canvas

Canvas Parameters are Optional, Refresh Rate will Default to 100ms, Width and Height will auto-size to the document's Specification's

game.create(refreshRate, Width, Height);

Setup Loop

let loop = () => {game.request(loop);}

game.request(loop);

Clearing Pixels

Pixels should be cleared everytime the loop runs so they can be redrawn
game.clear();
